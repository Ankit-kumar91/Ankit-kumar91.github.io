Personal portfolio website showcasing my experience as a Data Scientist with a background in Chemistry (PhD, IIT Bombay). Built with React, TypeScript, and Vite.

**Live:** [ankit-kumar91.github.io](https://ankit-kumar91.github.io/)


## Run Locally

**Prerequisites:** Node.js (v20+)

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

The app runs at `http://localhost:3000`.

## Build for Production

```bash
npm run build
```

Output goes to the `dist/` directory.
